PROMETHIUS DEVELOPMENT DOCUMENTATION

 

 

The Promisance Athius Game Engine (abbreviated as ‘PAGE’) is both a powerful and flexible web game engine. PAGE has a history extending back as far as five years, with most of its features having been eventually reconstructed from ground up. As a result, the methods of development have become uniform and far more efficient.

 

This documentation addresses the basis fundamentals of template design and the API.

 

 

 

1. API

 

HEADER

 

Check that the file is not being accessed directly

if ( !defined('PROMETHIUS') )

     die("Hacking attempt");

 

Synchronize with the rest of the game

include($game_root_path.'/header.php');

 

 

YOUR CODE HERE

 

You are now synchronized with the rest of the game.  You have access to all the shared variables.

 

Variables

 

$users

This contains the current user’s data from the game_players table

 

 

Functions

 

endScript(‘YOUR_ERROR’)

Use this if you want to stop the script and display formalized error message

 

doError(‘YOUR_ERROR’)

Use this if you want to still run the rest of your script, but display an error message in your template

 

loadEra($THEUSER)   

Use this to load the user’s regional data (‘era’ is derived from classic Promisance)

 

loadRace($THEUSER)  

Use this to load the user’s faction data (‘race’ is derived from classic Promisance)

 

takeTurns($TURNS, ‘PARAM’)

Use this to take turns as usual

 

saveUserData($THEUSER, ‘cash land food’)  

Use this to update the user’s data (separate the values with a space)

 

$tpl->assign('TPL_VAR', $VAR)  

Use this to share a variable or array with your template

 

FOOTER

 

We tell the game to begin rendering the page

TheStart('');

 

Specify the template that will contain the html data

$tpl->display('YOUR_HTML.tpl');

 

We tell the game to render the bottom of the page and end the script

endScript(‘’);
